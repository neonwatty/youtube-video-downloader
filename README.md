# a simple youtube_downloader app

A simple python app that lets you painlessly download youtube / shorts video files without needing to visit chintzy sites online.  Can be easily run locally.  Try it out now in your browser at [![HuggingFace Space](https://img.shields.io/badge/ðŸ¤—-HuggingFace%20Space-cyan.svg)](https://huggingface.co/spaces/neonwatty/youtube_downloader).

To run the app install the associated `requirements.txt` (in a venv) and run

```python
python -m streamlit run youtube_downloader/app.py
```
